# YouTube Get VideoInfo Tool

Youtube Data APIを利用して、指定したチャンネルの動画情報(URL、再生数等)やサムネイル画像を取得するアプリです。

## Requirements

- YouTube API
- docker
- docker-compose

## Setting

```bash
docker-compose up -d --build
```

## Usage

Access to <http://localhost:8501/>
